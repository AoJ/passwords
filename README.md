Requirements
------

Ark8/Passwords requires PHP 5.5 or higher.

- [Nette DI](https://github.com/nette/di)


Installation
------

The best way to install Ark8/Passwords is using [Composer](http://getcomposer.org/):

```sh
composer require ark8/passwords
```
